## 2. 配色システム

### 2-1. メインカラーパレット

| 用途 | カラー | 値 |
|------|--------|------|
| **プライマリブルー** | ロイヤルブルー | `rgb(40, 107, 203)` / `#286BCB` |
| **背景（ライトブルー）** | 薄い空色 | `rgb(232, 242, 250)` / `#E8F2FA` |
| **背景（ミッドブルー）** | やや濃いライトブルー | `rgb(212, 227, 240)` / `#D4E3F0` |
| **テキスト** | ダークグレー | `rgb(51, 51, 51)` / `#333333` |
| **白** | ピュアホワイト | `#FFFFFF` |
| **フッター背景** | ダークネイビー | 推定 `#1a2744` |

### 2-2. 配色の特徴

**「1色主義」** — グラデーションを使わず、ブルー1色（`#286BCB`）で統一。ヒーロー背景、ボタン、アクセント、すべて同じブルー。この潔さが「ブレない人」「信頼できる人」という印象を生む。

背景のバリエーションは**ブルーの濃淡のみ**:
- 白（`#FFF`）→ テキスト多めのセクション
- ライトブルー（`#E8F2FA`）→ お知らせ・実績セクション
- ブルー（`#286BCB`）→ ヒーロー・フッター上部

## 6. 写真の使い方（この サイト最大の特徴）

### 写真①：FVメイン（手を広げるポーズ）
- **構図**: やや下からのアングル、右寄せ配置
- **ポーズ**: 右手を大きく広げ、語りかけるジェスチャー
- **服装**: ネイビースーツ＋ブルーネクタイ＋白シャツ
- **表情**: 口を開けて話している瞬間（動きのある表情）
- **背景処理**: 完全に切り抜き、ブルー単色背景に配置
- **サイズ**: 画面の半分以上を占める大きさ

### 写真②：政策セクション（手を組む真剣な表情）
- **構図**: 正面やや上からのアングル
- **ポーズ**: 両手の指を組んで口元に当て、考え込むポーズ
- **服装**: **白シャツのみ（ジャケットなし）** — FVとの対比
- **表情**: 真剣、静かな決意
- **背景処理**: 切り抜き、白背景に配置
- **重ね方**: 背面に「青森新時代」の超巨大ブルータイポグラフィ

### 写真③：プロフィール（屋外カジュアル）
- **構図**: やや離れた位置から、環境込みで
- **服装**: Tシャツ（カジュアル・地域イベント参加時）
- **表情**: 自然な笑顔
- **背景**: 屋外の緑と空（地域の風景）
- **処理**: 切り抜きなし。**全幅写真**として使用
- **オーバーレイ**: 「プロフィール」テキストを写真の上に

### 写真の「3段階ストーリー」

```
FV（動）    → スーツ＋手を広げる   → 「この人は行動する」
メッセージ（静）→ 白シャツ＋手を組む   → 「この人は深く考える」
プロフィール（素）→ Tシャツ＋笑顔     → 「この人は人間味がある」
```

この **動→静→素** の三段構成が、サイト全体のストーリーテリングを支えている。

## 7. 装飾・グラフィック要素

### シンボルモチーフ
- FV背景に**青森のシンボル（りんごの花モチーフ）**を同系色のブルーで薄く重ねている
- 政策セクションにも同じモチーフが右側に配置
- **透過度**: 非常に薄い（背景の一部として認識される程度）

### 巨大タイポグラフィ装飾
- 「青森新時代」が政策セクションの**背景的な装飾テキスト**として機能
- 色はブルー（プライマリカラーの薄い版）
- 写真の背面に配置されている
