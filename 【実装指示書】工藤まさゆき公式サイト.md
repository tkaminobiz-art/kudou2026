## 1. デザインコンセプト

### 1-1. ひとことで言うと

**「宮下宗一郎サイト（miyashitasoichiro.jp）の空間設計・写真演出・品格」を、維新グリーンのカラーで再構築する。**

### 1-2. 目指す印象

- **県民に近い、頼れるパワフルな存在**
- 硬くない。でも軽くない。**「仕事できそうだけど話しかけやすい」**
- 清潔感＋重み＋躍動感の共存
- 53歳の経験値が伝わる落ち着きと、まだまだ動くエネルギー

### 1-3. 絶対にやらないこと

- ピンク系の色は一切使わない
- 古い政治家サイトのような堅苦しいレイアウト
- 写真なしのテキストだらけのページ
- ゴテゴテした装飾・影・グラデーション過多

## 2. デザイン仕様

### 2-1. カラーパレット

**「1色主義」** — 宮下サイトと同じく、メインカラー1色＋濃淡のみで構成する。

```css
:root {
  /* メインカラー（薄い若葉グリーン）*/
  --color-primary: #5BAD5E;

  /* 背景バリエーション */
  --color-bg-white: #FAFAF8;       /* わずかにウォームな白 */
  --color-bg-light: #EDF5EE;       /* 薄い緑背景（お知らせ・実績セクション用） */
  --color-bg-mid: #D4E8D6;         /* やや濃い緑（装飾用） */

  /* テキスト */
  --color-text: #2C2C2C;           /* 本文（やや深みのあるダークグレー） */
  --color-text-muted: #888888;     /* 日付・補助テキスト */

  /* ボタン */
  --color-btn-bg: #5BAD5E;         /* メインカラーと同じ */
  --color-btn-text: #FFFFFF;

  /* フッター */
  --color-footer-bg: #1a3324;      /* ダークグリーン */
  --color-footer-text: #FFFFFF;
}
```

### 2-2. タイポグラフィ

**Noto Sans JP 1ファミリーのウェイト違いで統一。** 宮下サイトがヒラギノ角ゴ3ウェイトで統一していた手法をGoogle Fontsで再現する。

```css
@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@400;500;700;900&family=Montserrat:wght@700;800&display=swap');

:root {
  --font-main: 'Noto Sans JP', sans-serif;
  --font-en: 'Montserrat', sans-serif;  /* 英語装飾・数字用 */
}
```

| 要素 | フォント | サイズ | ウェイト | 行間 | 字間 |
|------|---------|--------|---------|------|------|
| **ヒーロー名前** | Noto Sans JP | 80-100px (clamp) | 900 | 1.0 | 5px |
| **ヒーローふりがな** | Noto Sans JP | 24-32px | 400 | 1.0 | 1.5px |
| **セクション見出し** | Noto Sans JP | 36-50px | 700 | 1.7 | 2px |
| **メッセージ本文（中央揃え）** | Noto Sans JP | 18-20px | 500 | **2.6** | 1px |
| **通常本文** | Noto Sans JP | 15-16px | 400 | **2.2** | 0.7px |
| **ナビゲーション** | Noto Sans JP | 15-16px | 500 | — | 0.5px |
| **ボタン** | Noto Sans JP | 16px | 700 | — | 0.5px |
| **日付・補助** | Montserrat | 13-14px | 700 | — | 0.5px |
| **ティッカーテキスト** | Noto Sans JP | 48-56px | 700 | 1.0 | 1px |

> **重要**: 行間2.2（本文）と2.6（メッセージ）は宮下サイトの核心的な特徴。この「呼吸する行間」が品格を生む。絶対に縮めない。

### 2-3. 余白

## 6. アニメーション

| 要素 | アニメーション |
|------|-------------|
| **ティッカー** | CSS transform で左方向に無限スクロール（20-25秒/周） |
| **セクション表示** | Intersection Observer でスクロールに応じてフェードイン（opacity 0→1, translateY 20px→0, duration 0.8s） |
| **ボタンホバー** | opacity 0.85 + translateY(-2px) |
| **カードホバー** | translateY(-4px) + shadow拡大 |
| **ナビ固定** | スクロール時にstickyで上部固定 |

> アニメーション速度はすべて**0.6s以上**。宮下テイストの落ち着きを維持するため、速すぎるアニメーションは使わない。
