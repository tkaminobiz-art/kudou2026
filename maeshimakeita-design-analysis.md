## 1. サイト全体構成（ページマップ）

```
[ヘッダー（固定）]
  └─ LINE追加ボタン + ナビゲーション（カプセル型）+ ハンバーガーメニュー

[ヒーローセクション]
  └─ グリーングラデーション背景
  └─ 人物写真（切り抜き・右寄せ）
  └─ 名前テキスト（超巨大・白・縦書き混在）
  └─ キャッチコピー画像「パワーアップ岡山市！」
  └─ キーワードティッカー（無限横スクロール）

[Messageセクション]
  └─ 2カラム：左に見出し / 右にテキスト本文
  └─ 白背景・たっぷりの余白

[写真ギャラリー]
  └─ 横スクロールカルーセル（活動写真）

[Profileセクション]
  └─ 2カラム：左にテキスト＋数値カード / 右に全身写真
  └─ 数値ハイライト（得票数・マラソンタイム）

[議会での活動（Gikai）セクション]
  └─ 3カラムカード（政策・議会レポート・実績）
  └─ 薄いグレー背景カード

[市政報告（Report）セクション]
  └─ 横スクロールカルーセル（市政報告チラシ画像）
  └─ ページネーション矢印 + もっと見る

[お知らせ（News）セクション]
  └─ 2カラム：左にニュースリスト / 右にカード型記事

[お問い合わせ（Contact）セクション]
  └─ グリーングラデーション背景（角丸セクション）
  └─ 2カラム白カード（フォーム / LINE）

[フッター]
  └─ 名前＋所属 / 政党ロゴ×2 / コピーライト
```

## 2. 配色システム

### 2-1. メインカラーパレット

| 用途 | カラー | 値 |
|------|--------|------|
| **プライマリグリーン** | 鮮やかな若葉グリーン | `rgb(81, 161, 50)` / `#51A132` |
| **グラデーション起点** | ティールブルー | `rgb(72, 162, 196)` / `#48A2C4` |
| **グラデーション終点** | グリーン | `rgb(69, 169, 67)` / `#45A943` |
| **本文テキスト** | ダークグレー | `rgb(51, 51, 51)` / `#333333` |
| **ミュートテキスト** | 薄いグレー | `rgba(0, 0, 0, 0.4)` 程度 |
| **背景（白）** | ピュアホワイト | `#FFFFFF` |
| **背景（グレー）** | 薄いニュートラルグレー | `#F5F5F5` 程度 |
| **カード影** | 微細ドロップシャドウ | `rgba(0,0,0,0.2) 0px 2px 5px` |

### 2-2. グラデーション定義

```css
/* ヒーロー背景 & コンタクトセクション */
background: linear-gradient(135deg, rgb(72, 162, 196), rgb(69, 169, 67) 46%, rgb(69, 169, 67));
```

- **方向**: 135度（左上→右下）
- **構成**: ティールブルー → グリーン（46%地点でグリーンに完全移行）
- **使用箇所**: ヒーローセクション全体、お問い合わせセクション背景
- **印象**: 爽やかさ・若さ・信頼感を演出。政治家サイトとしては珍しい明るさ

## 4. レイアウト構造

### 4-1. ヒーローセクション

- **構成**: フルワイド・グラデーション背景
- **人物写真**: 右側に大きく配置、背景透過（切り抜き）処理
- **名前テキスト**: 「前島」が超巨大で左寄せ、「けいた」が縦書きで横に並ぶ（白色）
- **キャッチコピー**: 「パワーアップ岡山市！」画像（矢印グラフィック付き・黄色系）
- **ティッカーバー**: ヒーロー最下部に横一列、白カプセルがグリーン文字で無限スクロール
  - 内容: 「28歳」「岡山市議会議員」「史上最年少」「日本維新の会」「是々非々」「若者世代」「子育て世代」「スポーツ大好き」「フルマラソン2h37m」「早稲田大学」「IT人材」「英語も得意です」
  - gap: 8px, 13個の要素（クローン含む）

### 4-2. ナビゲーション

- **位置**: 固定（sticky/fixed）、右上
- **LINEボタン**: 独立した緑カプセルボタン（左側）
- **メインナビ**: 白背景のカプセル型コンテナ（角丸pill型）
  - 項目: トップ / お知らせ / プロフィール / 市政報告 / 議会レポート / お問い合わせ
  - ハンバーガーアイコン（緑丸ボタン）が右端
- **フォント**: こぶりなゴシック W6, 14px程度

### 4-3. コンテンツセクション共通パターン

```
セクション構成:
  ●英語ラベル (Rajdhani, 20px, 薄グレー + 緑●)
  日本語見出し (見出ゴMB31, 緑)
  ────
  コンテンツ（テキスト or カード or カルーセル）
```

- **セクション間余白**: 約 80-120px
- **コンテンツ最大幅**: 約 1200px（センタリング）
- **パディング左右**: 約 5%

### 4-4. カードコンポーネント

#### 議会活動カード（3カラム）
```
背景: #F5F5F5（薄いグレー）
角丸: 8px
影: rgba(0,0,0,0.2) 0px 2px 5px
パディング: 24-32px
構成: ●英語ラベル + 日本語タイトル + ボタンリンク
```

#### 数値カード（プロフィール内）
```
背景: 白
角丸: 8px
影: rgba(0,0,0,0.2) 0px 2px 5px
構成: ラベル（こぶりなゴシックW6）+ 大きな数値（Saira Condensed 50px）
```

### 4-5. ボタンスタイル

```
形状: カプセル型（pill）/ アウトラインスタイル
枠線: 薄いグレーボーダー
角丸: 24-30px（完全なカプセル型）
```

## 5. UIコンポーネント詳細

### 5-1. 無限スクロールティッカー

- **実装**: `<sd-loop-box>` カスタム要素 + `.loop-track` flex コンテナ
- **アニメーション**: CSS transform による無限横スクロール（左方向）
- **子要素**: 白背景カプセル（`border-radius: 16px`）にグリーンテキスト
- **フォントサイズ**: 72px（非常にインパクトが大きい）
- **gap**: 8px
- **クローン**: loop-clone クラスで要素を複製し、シームレスに繰り返し

### 5-2. 横スクロールカルーセル

- **使用箇所**: 写真ギャラリー、市政報告チラシ
- **ナビゲーション**: `< >` 丸ボタン（左右矢印）
- **写真ギャラリー**: 正方形〜縦長の写真がオーバーフローで横スクロール
- **市政報告**: チラシ画像カード + 日付 + タイトルテキスト

### 5-3. ハンバーガーメニュー

- **形状**: 緑色の丸ボタン内にハンバーガーアイコン
- **ナビバー右端に配置**
- **モーダルメニュー**: `.modal-1` で全画面メニュー展開（推定）

## 6. 空間設計（余白・リズム）

| 要素 | 余白量 |
|------|--------|
| ヒーロー上部パディング | 100px |
| セクション間 | 80-120px |
| セクションタイトル→コンテンツ | 32-48px |
| カード内パディング | 24-32px |
| コンテンツ左右パディング | 5% (約72px @1440px) |
| ティッカーアイテム間 | 8px |
| フッター上部余白 | 80px |

**全体の印象**: 余白はたっぷりと取られており「政治家サイトの堅さ」を感じさせない開放感がある。セクションごとの呼吸が効いている。

## 7. アニメーション・インタラクション

### 確認済み
- **ティッカー**: 無限横スクロール（左方向・一定速度）
- **カルーセル**: 矢印クリックでスライド切り替え
- **セクション遷移**: フェードイン系（Intersection Observer による scroll reveal の可能性）
- **ナビバー**: スクロール追従（固定）

### 推定される挙動
- ボタンホバー時の色変化・微細な移動
- カードホバー時の影拡大
- 写真ホバー時のスケール変化
- ページ読み込み時のフェードインアニメーション

## 8. レスポンシブ設計（推定）

- ティッカーのフォントサイズ縮小
- 2カラム → 1カラムへのスタック化
- 3カラムカード → 縦積み
- ナビゲーション → ハンバーガーメニュー完全展開
- 人物写真のサイズ・配置調整

## 9. 2026年トレンド要素の識別

### 該当するトレンド

| トレンド | 該当度 | サイト内での表現 |
|----------|--------|------------------|
| **キネティックタイポグラフィ** | ★★★ | ティッカーの72px巨大テキスト横スクロール。名前「前島けいた」の超巨大タイポグラフィ配置 |
| **動的グラデーション** | ★★☆ | ヒーロー＆コンタクトのティール→グリーン135度グラデーション（ただし静的） |
| **マイクロインタラクション** | ★★☆ | ボタンのカプセル型・矢印付きデザイン（ホバーエフェクトの可能性） |
| **ベントグリッド** | ★☆☆ | 3カラムカードが部分的にタイル的だが本格的ではない |

### 該当しないトレンド
- Liquid Glass（透過・ぼかしは不使用）
- 静寂のデザイン（むしろアクティブ・エネルギッシュ）
- ネオブルータリズム（太枠線・原色の荒削り感なし）
- ノイズテクスチャ（使用なし）

### このサイトのデザインアーキタイプ

**「クリーン・エネルギッシュ・政治家モダン」**

従来の政治家サイトの「堅い・古い・重い」印象を完全に払拭し、「若い・爽やか・オープン」な印象を作り出している。グリーンのグラデーションが「成長・若さ・環境」を象徴し、大胆なタイポグラフィとティッカーが「動き・スピード感・行動力」を表現。カプセル型のUI要素（ナビ・ボタン・ティッカーアイテム）が統一された柔らかさを生んでいる。

## 10. オマージュ実装時の重要ポイント

### 絶対に再現すべき要素（サイトの「核心」）

1. **ティール→グリーンのグラデーション背景**（ブランドカラーとして最重要）
2. **無限スクロールティッカー**（キーワードが白カプセルで流れる）
3. **人物切り抜き写真の大胆な配置**（背景透過・右寄せ）
4. **●英語ラベル + 日本語見出し**のセクション構造
5. **カプセル型（pill）UIの統一**（ナビ・ボタン・ティッカー）
6. **こぶりなゴシック + Rajdhani + Saira Condensed**のフォント三重奏
7. **数値を大きく見せるカード**（Saira Condensed 50px）
